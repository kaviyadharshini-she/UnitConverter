# Unit Converter App

# Project Name
Unit Converter

# Description
Unit Converter is a simple Android application developed using Java and XML in Android Studio.
It allows users to convert values between different measurement units quickly and accurately.

# Features
- Length Conversion
  - mm ↔ cm
  - cm ↔ m
  - m ↔ km

- Weight Conversion
  - mg ↔ g
  - g ↔ kg

- Temperature Conversion
  - °C ↔ °F
  - °F ↔ K
  - °C ↔ K

- Volume Conversion
  - mL ↔ L

- Easy-to-use interface
- Instant conversion results
- Supports decimal values

# Technologies Used
- Java
- XML
- Android Studio

# Project Structure
- MainActivity.java
- activity_main.xml
- AndroidManifest.xml
- res/
- drawable/
- values/

# How to Run
1. Open the project in Android Studio.
2. Sync the Gradle files.
3. Connect an Android device or start an emulator.
4. Click **Run**.
5. Enter a value, select the conversion type and units, then tap **Convert**.

# Output
The app converts the entered value into the selected unit and displays the result instantly.
